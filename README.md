## 🚀 Features
- **Aggressive Service Mapping:** Identify open ports, services, and versions with precision.
- **Vulnerability Hunting:** Automated NSE scripts for SQLi, XSS, and CSRF detection.
- **Discord Integration:** Real-time push notifications via Webhooks when "prey" is found.
- **Sleek UI:** Professional terminal experience powered by the `Rich` library.

## 🛠️ Prerequisites
- **Python 3.x**
- **Nmap Engine:** 
    - Linux/Termux: `sudo apt install nmap`
    - Windows: Install via [Nmap.org](https://nmap.org)
